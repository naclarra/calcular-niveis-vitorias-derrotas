# calculadora de partidas rankeadas

Projeto desenvolvido como parte do desafio da **DIO (Digital Innovation One)**, aplicando lógica de programação com JavaScript

## descrição do desafio

criar uma função que receba a quantidade de vitórias e derrotas de um jogador e calcule o saldo de partidas rankeadas (vitórias - derrotas), retornando o nível do jogador.

níveis definidos pelas vitórias:

| Vitórias | Nível |
|----------|------|
| Menor que 10 | Ferro |
| 11 – 20 | Bronze |
| 21 – 50 | Prata |
| 51 – 80 | Ouro |
| 81 – 90 | Diamante |
| 91 – 100 | Lendário |
| Maior ou igual a 101 | Imortal |

##  O que utilizei:

- JavaScript (Node.js)

## como Executar

1. Abra o terminal na pasta do projeto  
2. Digite:
```bash
node index.js
